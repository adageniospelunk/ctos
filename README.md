# ctosooa

CTO Sooatek tool - A command-line utility package.

## Installation

Install globally using npm:

```bash
npm install -g ctosooa
```

## Usage

```bash
ctosooa echo <message>
```

Example:
```bash
ctosooa echo "Hello World"
```

## Uninstallation

To remove the package:

```bash
npm uninstall -g ctosooa
```
